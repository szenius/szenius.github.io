    TAMCHIAK

    Tamchiak is a web application built on Ruby on Rails with HTML, CSS and some JavaScript/jQuery. With Tamchiak, groups of friends can create channels to collate their food orders with ease, then use the inbuilt service to split bills after.

    Contribution: My team of 2 (including myself) built this application from scratch in about two months. During this time, my partner and I conducted independent learning of Ruby on Rails, built the product and invited ~70 users to test our application. This project was done with no external help from mentors.
    Tamchiak GitHub Repository

    ROUND 2
     Round 2 is a web application built using Ionic framework and node.js. Built with the idea of reducing wastage of expired food in supermarkets and stores alike, Round 2 is a virtual supermarket, where items will be priced less towards their expiry dates. This entices users to purchase items nearer to their expiry dates - something that they usually will not do.

    Contribution: This project was done by a team of 4 (including myself). Other than coming up with the concept behind this project, I also helped to build a mockup of items uploaded by supermarkets. On top of that, I also added a function for users to search for nearby stores and items according to coordinates that they key in for their home location and their desired radius of proximity from this location.
    GitHub Repository
    TEMPO
     Tempo is a Java application that has both a text user interface and a graphical user interface. Tempo aims to allow users to manage their schedule with just a single line command. Other than the CRUD functions for events (with a start/end date and time), tasks (with deadline) and floating tasks (no deadline), Tempo also supports CRUD of recurring events and tasks. Other services that Tempo provide include natural language input and flexible input formats.

    Contribution: My team of 4 (including myself) built this application from scratch with advice from a teaching assistant. This project was done as part of a Software Engineering module and was completed in one semester. During the semester, we designed the architecture of the application and various component designs before working on the actual code.
    Download GitHub Repository

    PROMPTU
     Promptu is a Android mobile application that helps business professionals go on their business travel with great ease. Promptu features a calendar function to help users keep track of their schedule while on business trips. This calendar function allows for automatic extraction of important dates and documents from PDF files to be put in users's schedules on top of manual management of the calendar function. Promptu also features a carousel function which displays queries and requests from colleagues of the user on business travel who are back in the local company. Users can then respond to these queries and requests with a quick click on the YES or NO buttons. This will then trigger the sending of an email to the respective colleague.

    Contribution: Other than taking part in the idea creation process, I also built a mockup of the carousel function to display requests and queries from colleagues back in the local company. The mockup included the swiping left/right to view queries and requests and the clicking of YES/NO buttons. I also managed a Facebook page to market this application.
